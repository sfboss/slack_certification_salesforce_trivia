# Getting Started

This section gets you from "fresh clone" to "first answered question in Slack" in roughly
20 minutes, assuming you already have a Salesforce Dev Hub and a Slack workspace you can
install apps into.

The path:

1. [Prerequisites](prerequisites.md) — check tooling versions.
2. [Installation](installation.md) — clone, deploy to a scratch org, configure Named Credentials.
3. [Quick Start](quick-start.md) — import sample questions, install the Slack app, run `/certgame play`.

If you only want to read about the system before installing, jump to
[Architecture overview](../architecture/overview.md).
